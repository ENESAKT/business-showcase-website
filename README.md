# business-showcase-website

Responsive business showcase template — five pages of plain HTML & CSS, no frameworks, no build step.

**Live demo:** https://enesakt.github.io/business-showcase-website/

Pages: Home · About · Gallery · References · Contact

- Semantic HTML5, CSS custom properties, Grid/Flexbox, mobile-first
- Gallery ships with CSS placeholder tiles — swap each tile for your own `<img loading="lazy" srcset="…">`
- The original 2022 version carried ~25 MB of mixed downloaded imagery; it was purged from git history
  (copyright + repo size) and replaced with placeholders so the template is safe to reuse.
